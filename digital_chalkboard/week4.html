Tasks and guidance for February 04 — Simple JavaScript, Images, Footer

  1. Add some images to your index.html page
  2. Use Bootstrap classes to make sure the images behave.
  3. You can decide if you want an image to cover the entire width of the site or sit inside a column.
  4. You can decide if you want to use a Boostrap carousel or a simple div.
  5. Make sure the images adjust when the page resizes.
  6. Whatever approach you take, make sure the image is placed with intention.
  7. Add a bootstrap footer.
  8. Need a model for the footer? Check out the RCC site.
  9. We can take a moment to unpack the JQuery nav component from last class.

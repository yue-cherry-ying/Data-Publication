Digital Chalkboard Tasks

Add the new code to your own site.

**First: make sure you've caught up from all previous tasks**

  1. Create a Bootstrap container so the body of the page behaves well.
  2. Create a navigation section. Your choice of style.
  3. This site has an h1 heading. Can you figure out how to display the text in a regular div on a screen that is narrower than 768px?
  4. Insert a new container and row as the home for four columns.
  5. Add some text to the columns. Cf. https://www.lipsum.com/. Make sure the columns stack on a phone.
  6. Create an external CSS file to use with this page.
  7. Add a subtle background color behind the columns. Think about the best CSS selector to use!!
  8. Check out the Bootstrap documentation to find ways to add some spacing between the columns when they stack up.
  9. Be ready to use Inspect mode to check the responsive mode of the site.
  10. Use a @media query to change the text spacing in the columns to justify. On screens wider than the large breakpoint, the text can be display in a normal format. On screens smaller than the large breakpoint, the text in the columns should display as justified.
  11. Use CSS to create a drop cap for the first letter of each paragraph. HINT: you can use a pseudo-class called ::first-letter. You'll likely need the following CSS properties (tweak the values to your liking): color, font-family, font-size, line-height, padding-top, padding-right, padding-left. You'll also need float:left.
  12. Create a button on your page to launch a function.
  13. Create an external JavaScript file.
  14. Write a function to pop up a simple alert message.
  15. Use an onclick event to invoke your function.
  16. Add some images to your index.html page
  17. Use Bootstrap classes to make sure the images behave.
  18. You can decide if you want an image to cover the entire width of the site or sit inside a column.
  19. You can decide if you want to use a Boostrap carousel or a simple div.
  20. Make sure the images adjust when the page resizes.
  21. Whatever approach you take, make sure the image is placed with intention.
  22. Add a bootstrap footer.
  23. Need a model for the footer? Check out the RCC site.
  24. We can take a moment to unpack the JQuery nav component from last class.
